# PythonPortfolio

### Project 1: Board Games
**Summary**: This program allows recommends a random game to the user based on user input
#### **Key Features**:
- Recommends a game based on the type of mechanics
- Recommends a game based on the number of players
- Recommends a game based on the type of genre
- Recommends a game based on the game's playing time

### Project 2: Dog Breed
**Summary**: This program recommends a random dog to the user and find data of a specfic dog
#### **Key Features**:
- Recommends a dog based on their size
- Finds the temperment of the dog the user inputs
- Looks for dogs that with a similar input

### Project 3: Hacker
**Summary**: Analyzes data within the list of users, login time, protocol, etc
#### **Key Features**:
- Looks for a specfic word within the list
- Looks for data greater than 2000
- Looks for total amounts of password resets. 

### Project 4: Images
**Summary**: Recommends the user a vocational spot based on user input
#### **Key Features**:
- Ask the user what type of climate they like
- How they like to spend their time inside or outside
- Opens an image of the vocational spot

### Project 5: Simulation
**Summary**: Simulating the tortoise and the hare racing to the finish line. 
#### **Key Features**:
- Simulates the speed of the tortoise and the hare
- Simulates when the hare sleeps
- Keeps track of the amount of times the tortoise or the hare wins
